using turnin using turnin executive summary: this document contains information on using turnin to submit homework assignments. we'll be using the 'turnin' program to support electronic submission of programming assignments. the student interface to turnin is straightforward. important: you can must run turnin on the mscc machines for the lisp assignments; turnin on other machines (.u and .cs) will not submit your homeworks to the expected destination. suppose i've finished my problem set, and i'm ready to submit it. to turnin a set of files, one types 'turnin file1 file2 ..... filen'. turnin will ask you which class you are submitting the assignment for; answer cse341. here's a transcript of what happens when i use turnin to submit the files prob1.cl, bank.cl, and menus.lisp. turing_38% turnin bank.cl menus.lisp prob1.cl for this submission, you may enter the course information interactively. you are not subscribed to any valid course, here is a list of the courses you may use currently: cse341 ms100 cse401 cse473 enter course name? [cse341] cse341 compressing submitted files... please wait your files have been submitted to cse341, psl1 for grading. if you want to check and see which files you've actually submitted, you can give turnin the -v option: turing_39% turnin -v for this submission, you may enter the course information interactively. you are not subscribed to any valid course, here is a list of the courses you may use currently: cse341 ms100 cse401 cse473 enter course name? [cse341] cse341 rw-r--r-- 10107/117 3069 apr 4 17:15 1996 bank.cl rw-r--r-- 10107/117 5119 apr 4 14:51 1996 menus.lisp rw-r--r-- 10107/117 2022 apr 4 13:45 1996 prob1.cl turnin also takes a -c flag to specify the class. you can also use this interface. for example: turing_40% turnin -c cse341 bank.cl menus.lisp prob1.cl compressing submitted files... please wait your files have been submitted to cse341, psl1 for grading. turing_41% turnin -c cse341 -v rw-r--r-- 10107/117 3069 apr 4 17:15 1996 bank.cl rw-r--r-- 10107/117 5119 apr 4 14:51 1996 menus.lisp rw-r--r-- 10107/117 2022 apr 4 13:45 1996 prob1.cl you can run turnin as many times as you want, each time the new submission will completely replace the previous submission. we will use the time stamps of the submitted files to determine whether or not a submission is late. cse341-webmaster@cs.washington.edu (last update: 04/06/96 at 12am )
